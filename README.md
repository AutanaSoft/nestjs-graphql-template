## Installation

```bash
$ pnpm install
```

## Build

```bash
# build for production
pnpm build
```

## Running the app

```bash
# production mode
$ pnpm start

# development
$ pnpm start:dev
```

## Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test unit
$ pnpm test:unit
```

## Linting

```bash
$ pnpm lint

$ pnpm lint:fix
```
